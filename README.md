# Projeto-Organizador-IR

Planilha desenvolvida para organizar, registrar e consolidar informações financeiras necessárias para apoio no preenchimento da declaração de Imposto de Renda. O modelo centraliza dados do titular, informes bancários e receitas mensais, com consolidação automática e visualização gráfica.

---

## Objetivo

Facilitar a organização das informações financeiras do contribuinte, permitindo controle estruturado de rendimentos e dados bancários, além de consolidação automática para conferência e análise.

---

## Estrutura da Planilha

A solução é composta por quatro abas principais:

### Titular
- Dados cadastrais do contribuinte
- Informações de identificação
- Base de referência para o controle

---

### Informes Bancários
- Cadastro das instituições financeiras
- Registro de saldos e valores informados
- Apoio para conferência de informes de rendimentos

---

### Notas de Receitas
- Registro mensal de receitas
- Classificação por tipo:
  - Holerite (CLT)
  - PJ
- Valores recebidos
- Organização por mês

---

### Totais
- Consolidação automática dos valores
- Somatórios gerais
- Totais por tipo de rendimento
- Indicadores resumidos
- Gráficos automáticos de:
  - Distribuição de receitas
  - Evolução mensal

---

## Recursos Técnicos Utilizados

- Microsoft Excel
- Fórmulas de soma e consolidação
- Fórmulas condicionais
- Estrutura de tabelas
- Validação de dados
- Hiperlinks
- Gráficos dinâmicos

---

## Como Utilizar

1. Preencher a aba **Titular**
2. Inserir os dados na aba **Informes Bancários**
3. Registrar as receitas mensais na aba **Notas**
4. Consultar a aba **Totais** para visualizar consolidações e gráficos
